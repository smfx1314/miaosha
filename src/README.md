# miaosha

#### 介绍
前后端分离简单实现购物商城下单秒杀功能

#### 软件技术
+ springboot
+ mybatis
+ mysql
+ redis


#### 安装教程

1. 导入mysql数据表
2. docker一键部署
3. 更改server端口
4. Nginx部署前端页面

#### 使用说明

1. 注册账号正常访问
